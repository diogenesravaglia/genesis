# Projeto Genesis

## Descrição
Este repositório contém artefatos do projeto Genesis, incluindo scripts SQL e documentações relacionadas ao gerenciamento de oportunidades de vendas.

## Funcionalidades
- Consulta SQL para listar oportunidades de vendas com valor igual ou superior a 500,00 e status 'Concluído', trazendo nome, região e tipo do cliente.

## Estrutura dos Arquivos
- vendas_500k.sql: Script SQL que realiza a consulta conforme especificado na atividade NAP-29 do Jira.
- readme.txt: Este arquivo de documentação.

## Como utilizar
1. Clone o repositório:
   git clone https://github.com/diogenesravaglia/genesis.git
2. Utilize o script vendas_500k.sql em seu ambiente Oracle para obter as informações desejadas.
